### 4. [BACKUP_TIMER.md](./BACKUP_TIMER.md) — Резервный таймер и Интервальные таймеры
**Что внутри:**
- Как работает двухуровневая система времени (NTP + Backup)
- Работа расписаний БЕЗ интернета

**Ключевая фишка:**
```
Две системы таймеров:
• Сценарии — точное время суток (NTP/Backup Timer)
• Интервальные таймеры — работают ВСЕГДА на millis() без интернета!
```

## Типичные сценарии использования

### Сценарий 1: Первый запуск
1. Прочитайте [README.md](./README.md) — быстрый старт
2. Подключитесь к `http://192.168.4.1` веб-интерфейсу
3. Настройте WiFi, MQTT, датчики в UI
4. Готово!

### Сценарий 2: Управление через консоль
1. Прочитайте [COMMANDS.md](./COMMANDS.md) — все команды
2. Подключитесь через Serial Monitor (115200 бод)
3. Вводите команды:
   ```
   > status
   > sensors
   > light grow
   > scenario list
   ```

### Сценарий 3: Интеграция с Home Assistant
1. Прочитайте [API.md](./API.md) — REST API endpoints
2. Используйте `curl`, REST sensors, автоматизацию HA:
   ```bash
   curl http://192.168.1.100/api/fixture/set -d "red=100"
   ```
3. Готово для автоматизации!

### Сценарий 4: Расписания без интернета
1. Прочитайте [BACKUP_TIMER.md](./BACKUP_TIMER.md)
2. Включите расписаний (сценарии) в веб-интерфейсе
3. Даже без WiFi расписания будут работать! ✅

### Сценарий 5: Добавление нового датчика
1. Прочитайте раздел в [README.md](./README.md) — "Как добавить новый датчик"
2. Добавьте тип в `config.h`
3. Создайте драйвер `src/sensors/my_sensor.h`
4. Компилируйте и загружайте

## FAQ по документации

**Q: С чего начать, если я новичок?**  
A: [README.md](./README.md) → быстрый старт → веб-интерфейс.

**Q: Мне нужны команды для управления.**  
A: [COMMANDS.md](./COMMANDS.md) → примеры → шпаргалка.

**Q: Как интегрировать с Home Assistant?**  
A: [API.md](./API.md) → REST API примеры → curl команды.

**Q: Как сделать расписание БЕЗ интернета?**  
A: [BACKUP_TIMER.md](./BACKUP_TIMER.md) → двухуровневые часы.

**Q: Чем отличаются сценарии от интервальных таймеров?**  
A: Сценарии — в конкретное время суток (08:00). Интервальные — через заданный интервал (каждые 15 минут). Интервальные работают без интернета!

**Q: Работают ли интервальные таймеры без интернета?**  
A: Да! Интервальные таймеры используют millis() и работают всегда, даже без интернета. Подробнее: [BACKUP_TIMER.md](./BACKUP_TIMER.md)#интервальные-таймеры

**Q: Как добавить свой датчик?**  
A: [README.md](./README.md) → раздел "Как добавить новый датчик".

**Q: Какие датчики поддерживаются?**  
A: [README.md](./README.md) → таблица "Поддерживаемые датчики".
